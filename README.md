# 🏥 Home Healthcare Visit Scheduler

## 📖 Project Description

The Home Healthcare Visit Scheduler is a web-based healthcare management application developed using React JS. The system is designed to assist healthcare providers in organizing and managing home healthcare services efficiently. It provides a centralized platform for maintaining patient records, scheduling healthcare visits, tracking daily tasks, recording visit notes, managing notifications, viewing visit history, and monitoring system statistics through an administrative dashboard.

The application follows a component-based architecture using React and implements modern frontend development concepts such as routing, state management, event handling, and persistent data storage through LocalStorage.

---

## 🎯 Project Objectives

* To simplify patient record management.
* To schedule and organize healthcare visits efficiently.
* To maintain healthcare task checklists.
* To store and manage visit notes.
* To provide notifications and reminders.
* To maintain historical visit records.
* To provide administrative monitoring and reporting.
* To improve healthcare workflow management through a user-friendly interface.

---

## 🚀 Features

### 📊 Dashboard

Provides an overview of healthcare activities and displays key statistics.

### 👨‍⚕️ Patient Management

Allows users to add, view, and manage patient information.

### 📅 Visit Scheduler

Enables scheduling and management of home healthcare visits.

### ✅ Task Checklist

Helps track and manage healthcare-related tasks.

### 📝 Visit Notes

Allows recording and storing observations made during healthcare visits.

### 🔔 Notifications

Displays reminders and important healthcare-related updates.

### 📜 Visit History

Maintains records of previous healthcare visits.

### ⚙️ Admin Panel

Displays overall statistics such as total patients and scheduled visits.

### 💾 Local Storage Support

Stores application data locally to ensure persistence after page refresh.

## ⚙️ Installation and Execution

### Clone Repository


```bash

```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
